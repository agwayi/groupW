Attendance System API

Base URL: http://localhost:8000/api

Faculty Endpoints

GET /GetFaculty.php

Fetches all faculties from the database.

Success Response (200 OK): 
{
    "status": "success",
    "data": [
        { "faculty_id": 1, "name": "Engineering" },
        { "faculty_id": 2, "name": "Science" }
    ]
}

POST /AddFaculty.php

Creates a new faculty.

Important: Send data as FormData (x-www-form-urlencoded), NOT as raw JSON.
Parameter Type Description
facultyname String The name of the faculty to add.
Success Response (201 Created): 
{
    "status": "success",
    "message": "Faculty added successfully"
}

PUT /EditFaculty.php

Updates an existing faculty name.

Important: Send data as a raw JSON string with the header Content-Type: application/json.
Request Body (JSON): 
{
    "faculty_id": 1,
    "name": "Applied Sciences"
}
Success Response (200 OK): 
{
    "status": "success",
    "message": "Faculty updated successfully"
}

Department Endpoints

GET /GetDepartment.php

Fetches all departments from the database.

Success Response (200 OK): 
{
    "status": "success",
    "data": [
        { "department_id": 1, "name": "Computer Science", "faculty_id": 1 }
    ]
}

POST /AddDepartment.php

Creates a new department linked to a faculty.

Important: Send data as FormData (x-www-form-urlencoded), NOT as raw JSON.
Parameter Type Description
name String The name of the department.
faculty_id Integer The ID of the parent faculty.
Success Response (201 Created): 
{
    "status": "success",
    "message": "Department added successfully"
}

PUT /EditDepartment.php

Updates an existing department.

Important: Send data as a raw JSON string with the header Content-Type: application/json.
Request Body (JSON): 
{
    "department_id": 1,
    "name": "Software Engineering",
    "faculty_id": 1
}
Success Response (200 OK): 
{
    "status": "success",
    "message": "Department updated successfully"
}
